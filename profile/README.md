## 자초단 멤버 소개 👋

### 자초단
자바 초보단 이라는 이름으로 딴도, 앙큼, 환장 외의 여러 명이 진행한 스터디

### 어쩌다 프로젝트 팀 이름이 자초단이 되었는가
자초단 1기, 스터디가 끝나고 딴도는 프로젝트를 해야겠다고 생각했다 </br>
그 때, 인턴이 종료된 수댕과 현업 짱짱 프론트맨 졸려를 발견하고 </br>
그들을 납치하게 되는데...
